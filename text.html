Blog Questions & Answer

1. Discuss the scope of var, let, and const ?

Ans:-
  • var : var is Function Scoped.
  • let : let is Block Scoped.
  • const : const Declarations are Block Scoped, and const declares a variable with a constant value.

    • 2. Tell us the use cases of null and undefined ?

    Ans:-
  • null : Null means an empty value or a blank value.
  • undefined : Undefined means the variable has been declared, but its value has not been assigned.
